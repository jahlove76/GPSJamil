# GPS Jamil

Application mobile simple de localisation GPS.

## Fonctionnalités
- Affiche la position GPS actuelle (latitude et longitude)
- Demande automatiquement la permission GPS
- Bouton pour mettre à jour la position
- Interface simple et intuitive

## Technologies
- React Native avec Expo
- Expo Location pour le GPS

## Installation
```bash
npm install
npm start
```

## Utilisation
1. Scannez le QR code avec l'application Expo Go
2. Autorisez l'accès au GPS
3. Votre position s'affiche automatiquement
